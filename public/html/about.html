About mySpotifyApp

This is an interview project. The requirements (intentionally paraphrased) include:

  • Build a web app that uses the Spotify API. The user should be able to:
    • Search for artists
      • Display a dropdown with search suggestions while the user types
    • Show a list of albums when the user clicks on an artist
    • Show the details of the album, including tracks and duration, when the user clicks on an album
  • Use ReactJS (not done)
    • See note below under What's next
  • Use any other framework, standard code, etc. This project uses:
    • Bootstrap 4
    • jQuery
    • Node.js
    • Autocomplete library from www.algolia.com
    • Example login code from Spotify
  • Preserves browser state, i.e. history buttons work (sort of works)
    • This sort of works, based on a made-up-hacked-up solution I came up with based on hashes.
    • The correct solution requires the use of router code.
  • Bonus
    • Compatible with all modern browsers
      • Intentionally used Bootstrap, which is designed for that purpose
    • Use of SASS/LESS and BEM would be appreciated (not done)
    • UX evaluated
      • Tried to make this app flow well without specifically copying any other examples on Spotify's developer site
      • Better colors could be used
      • Only retrieves the first x search matches, needs to implement "get next"
  • What's next
    • Port or rewrite with ReactJS
      • Since I have not used ReactJS before and had a short deadline, I decided to get something working using methods I do know, then port to ReactJS.
    • Add router code or use a framework
      • My last use of router code was about four years ago. The fact that this is what I was missing finally occurred to me as I was nearing my deadline.
    • Make better use of Bootstrap
      • Dev and testing were done with Chrome, alignment needs to be improved in Edge
      • Code needs to be added for collapsing the header on mobile
      • Remove the CSS short-cut hacks I added just to finish. That is not proper design.
